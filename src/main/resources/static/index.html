Bem-vindo(a) ao Bank API

Clientes

  • Visualizar Todos os Clientes: /json/customers

  • Visualizar um Cliente por ID: /json/customers/umId

  • Criar Conta para um Cliente: /json/accounts/create

    • Exemplo de Body: {"name": "Joaquim José", "cpf": "000.000.000-00"}

Contas

  • Visualizar todas as Contas: /json/accounts

  • Visualizar um Número de Conta: /json/accounts/umNumDeConta

  • Criar novo Cliente: /json/accounts/create/idCliente

    • Indicar o Id do cliente no último fragmento URI.

Transações

  • Efetuar Deposito em uma Conta: /json/transactions/deposit

    • Exemplo de Body: {"acNumber": "umNumDeConta", "amount": 500.0}

  • Efetuar Saque em uma Conta: /json/transactions/withdraw

    • Exemplo de Body: {"acNumber": "umNumDeConta", "amount": 100.0}

  • Visualizar todas as Transações de uma Conta: /json/transactions/umNumDeConta
